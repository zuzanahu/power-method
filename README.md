# power-method

## How to run
#### Prerequisites 
Check if you have Python installed on your system. 

python3 --version

Download the matrix "higgs-twitter.mtx" from https://sparse.tamu.edu/SNAP/higgs-twitter and have it in the same directory as main.py.

### 1. Create virtual environment
python3 -m venv .venv

### 2. Activate it
source .venv/bin/activate

(Windows: .venv\Scripts\activate)

### 3. Install dependencies
pip install -r requirements.txt

### 4. Run
python main.py
